# 策略回测使用说明

## 入口

在「策略组合」页面选择某个策略后，切到「业绩」标签，点击右上角「回测」。

## 回测参数

- 开始日期 / 结束日期：回测区间
- 初始本金：默认使用当前策略本金，可手动修改
- 费率：默认继承策略费率
- 再平衡模式：
  - `不再平衡`：仅初始建仓后持有
  - `偏离阈值`：当当前权重偏离目标权重超过阈值触发
  - `固定周期`：按固定交易日周期触发
  - `阈值 + 周期`：满足任一条件触发
- 偏离阈值(%)：仅在阈值模式/混合模式生效
- 固定周期(交易日)：仅在周期模式/混合模式生效
- 仅用本地缓存数据：开启后仅使用本地 `fund_history` 数据，不主动联网补齐

## 交易单位规则

- ETF：默认按 100 份整数手处理
- 普通基金：默认按 1 份处理

## 输出结果

- 资本结果：初始本金、期末市值、总收益、收益率
- 指标结果：年化收益、年化波动、Sharpe、Alpha、Beta、Calmar、信息比率、最大回撤
- 再平衡统计：调仓次数、成交笔数、换手金额、手续费
- 曲线：组合 / 基准 / 超额

## AI 深度回测分析

回测结果下方提供 AI 分析模块，支持两种预制风格：

- `硬核审计风格`：更强调回撤约束、收益质量与执行效率
- `稳健顾问风格`：更强调策略可解释性与长期执行建议

输出包含：

- 一句话结论
- 风险等级
- 综合分析报告
- 优势列表
- 风险点列表
- 动作建议列表

## 缓存与性能

回测结果使用参数哈希 + 数据版本标签做缓存：

- 同参数重复回测可快速命中缓存
- 当底层历史净值更新后，会自动失效并重算
